# My Learnings

A personal knowledge base for Salesforce development — Apex, LWC, integrations, CPQ/Billing, PSA, and architecture. Built to be searchable under pressure: interviews, client calls, or just needing a fast answer.

## How to use this

Use the **search bar** (top right) to find anything by keyword. Type "pricing waterfall", "JWT Bearer", "sharing", "triggers", "PKCE" — it searches across all pages instantly.

Or browse by topic using the sidebar on the left.

## Adding new content

Every new thing learned goes into the right topic folder as a `.md` file. The sidebar updates automatically. No special tooling needed — just markdown.
